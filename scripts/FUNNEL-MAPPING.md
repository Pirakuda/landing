| # | Роль в воронке | AIDA-этап | scrFull по умолчанию |

## 5. Правила для AI-агента «Архитектор»

1. **Всегда 7 уровней.** P1-P7 не меняются по количеству и порядку.
2. **Семантическая роль выбирается** из маппинга по `business_profile`. Если профиль `custom` — агент определяет роли из брифа.
3. **Количество экранов** — в пределах min/max из таблицы 3.1. Агент определяет точное число из объёма контента в брифе.
4. **CTA обязательны** на P1, P3 (каждый экран), P5, P7. На остальных — опционально.
5. **img_pos чередуется** внутри карусели (left → right → left) для визуального ритма, кроме P1 и P7 (center).
6. **scrFull = "full"** для P1 и P7 всегда. P5 и P6 — на усмотрение агента.
7. **rating = "true"** только на P1 (Hero) если у бизнеса есть рейтинг.
8. **phone = "true"** только на P1 и P7.
9. **Палитра** подбирается из рекомендаций раздела 4, финальный выбор — за агентом «Дизайн-директор».
10. **Язык контента** определяется параметром `lang` из команды `/newPage//domain//lang`.
